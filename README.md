# 🍔 The Kitchen Flava

> Sitio web de comidas rápidas — diseño premium, carrito integrado y formulario de pedido.

![Demo del sitio](images/Animation2.gif)

---



## ✨ Características

- Menú completo con filtros por categoría (Principales, Acompañamientos, Bebidas)
- Carrito lateral con control de cantidad por producto
- Formulario de pedido integrado como modal (sin páginas externas)
- Carrusel automático de imágenes
- Diseño responsive (móvil y escritorio)
- 100% imágenes propias — sin dependencias externas
